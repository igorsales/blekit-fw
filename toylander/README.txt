README

Outputs
==================

O1 = Left Turn Signal  - B(L)ue wire
O2 = Right Turn Signal - G(R)een wire
CH1 = PWM Motor control from Accelerator - White/Yellow wire from ESC
CH2 = Headlights - Yellow wire to light control board
CH3 = Rear lights - Orange wire to light control board


Inputs
==================

I1 = Forward Selection (Active Low)
I2 = Reverse Selection (Active Low)
I3 = Left Turn Signal Switch (Active Low)
I4 = Right Turn Signal Switch (Active Low)
I5 = Headlights Switch (Active Low)
I6 = Brakes?
S1 = Accelerator pedal resistence
S2 = Brake pedal pedal resistence
